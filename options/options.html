Context Weaver - Options

Context Weaver enhances your browsing experience by allowing you to transform text using powerful AI models. First, you will need to input your generative AI API key(s), and reload the extension. Then, open a new tab, highlight some text, and open the extension popup. Choose a transformation archetype and an AI model, and click "Weave". The transformed text will appear in the popup, ready for you to use. Enter your API keys below to enable AI processing.

API Key Configuration

Get your API keys from the respective AI model providers: OpenAI, DeepSeek, and xAI.

WARNING: Input API keys at your own risk. Context Weaver is not responsible for your API key security. API keys are stored locally in chrome.storage.sync. Context Weaver itself collects no data, but that does not make this extension secure. Be aware of the Terms of Service and Privacy Policy of the assorted AI services Context Weaver integrates with.
